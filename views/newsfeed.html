Overview

A News Feed website that is taken from Reddit's layout

View it Here!

Objective

The objective of this project is to be able to re-create Reddit's news feed layout and be able to create an account and posts as well.

Solution

By using Laravel, I am able to create authentications and easily seed + migrate the database. To be able to really make it identical to Reddit, I had to inspect Reddit's page a lot to get the right hex codes and icons. With Laravel, I am also able to create relationships which allowed me to have "Upvotes" + "DownVotes" functions.
